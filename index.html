4.- En este momento resulta importante poner en claro lo que queremos calcular. Usaremos tres variables aleatorias: X para la edad, Y para la temperatura y Z para el tipo de bebida. Cuando un cliente llega con el robot, por observaci√≥n se obtiene el valor de X y Y, mismas que servir√°n para predecir el valor de Z.

Entonces, lo que el robot debe calcular es la probabilidad de que la persona vaya a solicitar una bebida del tipo Z dado que conoce los valores de X y Y. Es decir, se calcula:

\[P(Z|X \cap\ Y) \]

Pongamos un ejemplo concreto. Imaginemos que llega un nuevo cliente. El robot observa que se trata de un adulto mayor y que la temperatura del d√≠a es fr√≠a. Entonces el robot sabe que ùëã=ùëöùëéùë¶ùëúùëü, y que ùëå=ùëìùëü√≠ùëú. Dadas estas observaciones, el robot debe anticipar lo que es m√°s probable que pida el usuario.

Conociendo que el cliente es un adulto mayor y que se trata de un d√≠a fr√≠o, ¬øes m√°s probable que el cliente pida una bebida fr√≠a o una caliente? Entonces debemos de calcular ambas probabilidades:

\[P(Z=fr√≠a|X = mayor \cap\ Y=fr√≠o) \]

\[P(Z=caliente|X = mayor \cap\ Y=fr√≠o) \]

El robot elige la mejor opci√≥n para ofrecer (bebida fr√≠a o bebida caliente) dependiendo de cu√°l de las probabilidades anteriores es mayor (elige la probabilidad m√°s grande). Es decir:

\[Sugerencia = argmax_{i \in \{fr√≠a,caliente\}}{P(Z= i|X = mayor \cap\ Y=fr√≠o) }\]

Esta √∫ltima l√≠nea es el clasificador bayesiano ingenuo para este problema, que acabar√° regresando como respuesta una sugerencia, que puede tomar alguno de los valores del conjunto {ùëìùëü√≠ùëé,ùëêùëéùëôùëñùëíùëõùë°ùëí} y que corresponde al argumento que genera el mayor resultado al sustituir ùëñ en la expresi√≥n ùëÉ(ùëç=ùëñ|ùëã=ùëöùëéùë¶ùëúùëü‚à©ùëå=ùëìùëü√≠ùëú).

El reto ahora es c√≥mo calcular estas probabilidades condicionales. Para ello, utilizaremos el teorema de Bayes.

\[P(Z|X \cap Y) = {P(Z)P(X \cap Y|Z)\over P(X \cap Y)}\]
