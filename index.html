Definición del Problema

En una cafetería se instala un robot que atenderá a los clientes. Se desea que el robot tenga iniciativa al atender al cliente, y recomiende productos antes de que se lo soliciten. El robot cuenta con un sistema de visión que le permite calcular la edad aproximada del cliente y cuenta con acceso a la web para saber la temperatura actual. A partir de la edad del cliente, y de la temperatura del día, el robot le recomendará una bebida fría o una bebida caliente.


¿Cómo va a tomar esta decisión el robot? Es aquí donde entra la probabilidad y estadística. El robot contará con datos previos sobre compras realizadas. En cada compra se registró la edad del cliente, la temperatura del día, y el tipo de bebida que compró (fría o caliente). Estos datos se encuentran en un archivo de texto y servirán para que el robot aprenda lo que debe hacer cuando llegue un nuevo cliente. Este aprendizaje lo haremos con un algoritmo bastante sencillo que se conoce como clasificador ingenuo de Bayes, y que consiste en calcular probabilidades condicionales con ayuda de la regla de Bayes.


En este caso, nuestras variables predictoras serán la edad del cliente y la temperatura del día. Se dice que el método que usaremos es ingenuo, porque utiliza la hipótesis de que estas variables predictoras son independientes entre sí. A partir de estas variables el robot debe predecir el tipo de bebida adecuado para el cliente.

4.- En este momento resulta importante poner en claro lo que queremos calcular. Usaremos tres variables aleatorias: X para la edad, Y para la temperatura y Z para el tipo de bebida. Cuando un cliente llega con el robot, por observación se obtiene el valor de X y Y, mismas que servirán para predecir el valor de Z.

Entonces, lo que el robot debe calcular es la probabilidad de que la persona vaya a solicitar una bebida del tipo Z dado que conoce los valores de X y Y. Es decir, se calcula:

\[P(Z|X \cap\ Y) \]

Pongamos un ejemplo concreto. Imaginemos que llega un nuevo cliente. El robot observa que se trata de un adulto mayor y que la temperatura del día es fría. Entonces el robot sabe que 𝑋=𝑚𝑎𝑦𝑜𝑟, y que 𝑌=𝑓𝑟í𝑜. Dadas estas observaciones, el robot debe anticipar lo que es más probable que pida el usuario.

Conociendo que el cliente es un adulto mayor y que se trata de un día frío, ¿es más probable que el cliente pida una bebida fría o una caliente? Entonces debemos de calcular ambas probabilidades:

\[P(Z=fría|X = mayor \cap\ Y=frío) \]

\[P(Z=caliente|X = mayor \cap\ Y=frío) \]

El robot elige la mejor opción para ofrecer (bebida fría o bebida caliente) dependiendo de cuál de las probabilidades anteriores es mayor (elige la probabilidad más grande). Es decir:

\[Sugerencia = argmax_{i \in \{fría,caliente\}}{P(Z= i|X = mayor \cap\ Y=frío) }\]

Esta última línea es el clasificador bayesiano ingenuo para este problema, que acabará regresando como respuesta una sugerencia, que puede tomar alguno de los valores del conjunto {𝑓𝑟í𝑎,𝑐𝑎𝑙𝑖𝑒𝑛𝑡𝑒} y que corresponde al argumento que genera el mayor resultado al sustituir 𝑖 en la expresión 𝑃(𝑍=𝑖|𝑋=𝑚𝑎𝑦𝑜𝑟∩𝑌=𝑓𝑟í𝑜).

El reto ahora es cómo calcular estas probabilidades condicionales. Para ello, utilizaremos el teorema de Bayes.

\[P(Z|X \cap Y) = {P(Z)P(X \cap Y|Z)\over P(X \cap Y)}\]


5.- Empezaremos calculando la distribución de Z, que es la probabilidad a priori. Esto corresponde a la probabilidad de que un cliente solicite una bebida del tipo Z (sin tomar en cuenta la información sobre la edad de la persona o la temperatura del día). Esta distribución la obtenemos contando a partir de nuestros datos.

\[P(Z= fría) = {\#clientes\ que\ compraron\ bebida\ fría \over \#\ clientes}\]

\[P(Z= caliente) = {\#clientes\ que\ compraron\ bebida\ caliente \over \#\ clientes}\]

Ejemplo. - A continuación, se muestra cómo calcular la distribución de Z con ayuda de Python.

# Cálculo de la distribución de Z # La distribucion se almacena en una lista, donde la posición 0 corresponde a P(Z=fría) y la posición 1 a P(Z=caliente) # Recordar que estamos presentando fría con 0, y caliente con 1 # Inicio del conteo en cero PrioriZ=[0,0] for i in range(0,n): PrioriZ[Tabla.Bebida[i]]+=1 # Muestra el conteo de cuantas bebidas de cada tipo hay print('Numero de bebidas según su tipo: ', PrioriZ) # Divide entre el número de clientes para obtener la distribución de probabilidad for i in range(0,2): PrioriZ[i]/=n print('Distribución de probabilidad de z: ', PrioriZ)

Ejemplo 1

Suponer que los datos ingresados son 𝑋=𝑚𝑎𝑦𝑜𝑟,𝑌=𝑓𝑟í𝑜.

Se tiene que calcular utilizando el teorema de Bayes:

\[P(Z=fría|X = mayor \cap\ Y=frío) \]

\[P(Z=caliente|X = mayor \cap\ Y=frío) \]

Por fortuna, todos los datos requeridos ya se encuentran en nuestras tablas:

\[P(Z=fría|X = mayor \cap\ Y=frío) = {P(Z=fría)P(X = mayor \cap Y = frío | Z = fría)\over P(X = mayor \cap Y = frío)}\]

\[ = {(0.339)(0.0235) \over 0.077} = 0.1034 \]

\[P(Z=caliente|X = mayor \cap\ Y=frío) = {P(Z=caliente)P(X = mayor \cap Y = frío | Z = caliente)\over P(X = mayor \cap Y = frío)}\]

\[ = {(0.661)(0.1043) \over 0.077} = 0.8953 \]

Con Z=fría se obtiene 0.1034. Con Z=caliente se obtiene 0.8953. Se elige el valor de Z que da el máximo resultado. Por lo tanto, la sugerencia debe ser una bebida caliente.

Con Z=fría se obtiene 0.9637. Con Z=caliente se obtiene 0.0354. Se elige el valor de Z que da el máximo resultado. Por lo tanto, la sugerencia debe ser una bebida fría.
